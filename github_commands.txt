# Commands to run after creating GitHub repository

# 1. Add remote origin (replace with your actual repository URL)
git remote add origin https://github.com/Manupati-Suresh/wine-classification-app.git

# 2. Push to GitHub
git push -u origin main

# 3. Verify the push worked
git status

# Alternative: If you get authentication errors, you might need to use a personal access token
# Go to GitHub Settings > Developer settings > Personal access tokens > Generate new token
# Then use: git push https://YOUR_TOKEN@github.com/Manupati-Suresh/wine-classification-app.git main
